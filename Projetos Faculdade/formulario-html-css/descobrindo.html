- Tag <footer></footer>

- A referida tag, é utilizada para fazer uma nota de rodapé, notmalmente utilizado no final do site para mostrar direitos reservados, mapa do site ou voltar ao topo.

- Tag <aside></aside>

- A tag representa um conteudo que é relacionado com o conteudo do seu entorno na pagina que pode ser separado dos demais.

- Tag <nav></nav>

- A tag define um conteúdo de navegação. Utilizada normalmente para criar menus.

- Tag <video></video>

- Essa tag serve para indicar a inserção de um vídeo. Com isso, devemos colocar alguns atributos dentro da tag para poder ativar o controle, a principal dela é o controls. Posteriormente, incluímos a tag <source> com o atributo src, onde receberão o local em que está o vídeo e o tipo do vídeo.
